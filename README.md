## Overview

This is my personal developer portfolio — a production-grade web application built to showcase my work and serve as a direct advertisement of my capabilities to businesses looking for a professional website. It is designed to be immersive, interactive, and fully responsive across all devices.

The site opens with a **cinematic laptop intro sequence**: click the laptop, watch the lid open, pass a facial recognition scan, and zoom through the screen into the portfolio. Every section is animated with Framer Motion and built with a dark luxury editorial aesthetic inspired by award-winning agency sites.

**Live at → [ivanfolio.sbs](https://ivanfolio.sbs)**

## Features

| Feature | Description |
|---|---|
| **Cinematic Intro** | 3D laptop opening, facial recognition scan, zoom portal entrance |
| **Oversized Hero** | Full-viewport-width "IVAN" display text with letter-by-letter animation |
| **Orbiting Language Logos** | 18 coding language icons orbit the profile photo in two counter-rotating rings |
| **Editorial Sections** | Numbered sections `(01)–(05)` in withradiance.com editorial style |
| **Accordion Services** | Numbered service rows that expand on click with detailed descriptions |
| **Project Showcase** | Full-bleed project cards for Huitlacoche Taqueria and Brooklyn MentorHub |
| **Animated Skills** | Skill bars with hover interactions and a scrolling tech marquee |
| **Contact Form** | Direct mailto form forwarding to personal email |
| **Custom Cursor** | Magnetic cursor with cyan glow that follows mouse movement |
| **Fully Responsive** | Mobile, tablet, and desktop layouts with fluid typography |

## Getting Started

```bash
# Clone the repository
git clone https://github.com/ivanexee/ivan-portfolio.git
cd ivan-portfolio

# Install dependencies
pnpm install

# Start the development server
pnpm dev
```

The dev server will start at `http://localhost:3000`.
